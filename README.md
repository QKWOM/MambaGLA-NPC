# MambaGLA-NPC

## Environment install

Clone this repository and navigate to the root directory of the project.
```
git clone https://github.com/QKWOM/MambaGLA-NPC.git

cd SegMamba
```

### Install causal-conv1d
```
cd causal-conv1d

python setup.py install
```

### Install mamba
```
cd mamba

python setup.py install
```

### Install monai
```
pip install monai
```
### Simple test
```
python test.py
```
